# New2RoboCup

A beginner-friendly collection of links and notes to help you quickly find the right entry point into the RoboCup ecosystem. Use this repository as a starting hub to explore the different RoboCup leagues and choose a track that matches your interests (soccer, rescue, @Home, industrial, or junior). Contributions are welcome—feel free to propose improvements or add missing resources via issues or pull requests.

## RoboCup leagues (quick overview)
RoboCup is an international robotics competition with multiple leagues that focus on different research and application areas—from autonomous robot soccer to search-and-rescue and assistive home robotics. The links below point to the official league pages (or the official RoboCupJunior site) so you can jump directly to rules, team information, and resources. Pick a league that fits your hardware, software, and learning goals, then start exploring.

## How to use this repo
1. **Choose a league** from the list above.
2. Open the official league page and look for: rules, field/arena specs, simulation environments (if available), and team documentation.
3. Track what you learn here (notes, additional links, or setup instructions) so the next newcomer has an easier start.
